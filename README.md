PS and examinePS are the core modules for the piecewise smoothing method.

• PS: can be run separately, and provides 1DPS and 2DPS predictions and maps. Example code to run it can be found in the PS.Rmd file.

• examinePS: can be run alongside the PS module to provide additional model assessment. Example code to run the two modules can be found in the examinePS.Rmd file.

It will be neccesary to install the “SpaDES” package before running any modules.
